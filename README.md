# HUGS Paper Homepage

This repository contains a static project page for the HUGS paper.

## Structure

- `index.html`: main project page template
- `assets/css/style.css`: page styling
- `assets/js/main.js`: real-world rollout layout tabs and video path switching
- `assets/images/`: teaser images, method figures, result figures
- `assets/videos/overview/`: overview video
- `assets/videos/real_world/`: real-world rollout videos
- `paper/`: optional local paper assets; the homepage currently links to arXiv
- `assets/data/`: optional downloadable data or metadata

## Usage

Fill in the empty slots in `index.html`, then replace files in `assets/` with the final paper materials.

The current template follows a typical research project-page layout:

- Hero title, authors, affiliations, and project links
- Overview video
- Abstract, key metrics, and teaser figure
- Real-world rollout sections with layout tabs
- Simulation figure and quantitative table
- Method figure and method cards
- BibTeX citation
